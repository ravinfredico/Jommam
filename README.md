# Jommam Map Application

This project is a map-based web application focused on food restaurants and stalls. It integrates the Google Maps API to display locations, and each location can feature social media content (such as TikTok videos) related to the venue.

## Features
- Interactive map with food locations
- Google Maps API integration
- Social media (e.g., TikTok) embeds for each location

## Stack
- Web application (suggested: JavaScript/TypeScript, React or similar)
- Google Maps API
- Social media embed support

## Setup
1. Install dependencies (if any)
2. Configure Google Maps API key
3. Run the development server

## Development
- Add new locations with metadata (name, description, coordinates, social links)
- Customize map markers and info windows
